Tutorial cepat
  1. Tap tombol + untuk input transaksi.
  2. Pakai tab Cepat (AI) untuk parsing kalimat bebas.
  3. Pakai Foto Struk untuk OCR dan simpan otomatis.
  4. Tap transaksi untuk edit / hapus.
  5. Untuk filter/search/periode, klik Periode.
